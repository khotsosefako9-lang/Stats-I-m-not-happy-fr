Probability & Statistics

Core definitions tested in Test 1 every year.

Core Definitions

TermDefinition
Random VariableA variable whose value is the numerical outcome of a random experiment. You cannot know its value before the experiment runs.
Sample SpaceThe complete set of ALL possible outcomes. Coin flip: {H, T}. Die: {1,2,3,4,5,6}.
Type I ErrorRejecting H₀ when it is actually TRUE. False positive. Probability = α.
Type II ErrorFailing to reject H₀ when it is actually FALSE. False negative. Probability = β.
Statistical IndependenceX and Y are independent if f(x,y) = f(x) × f(y) for ALL combinations of x and y.
Discrete VariableTakes countable values with gaps. Use PMF: P(X=x).
Continuous VariableTakes any value in a range. Use PDF: f(x). Probability = area under curve.

Properties of Expected Value

★
Exam trigger: "State any TWO properties of expected value." Know at least 4 so you can pick the easiest two.

⚠

Section B — Theory

Theoretical Distributions

Test 1 Q4 type: "State two properties and sketch the distribution."

Normal Distribution
  • Symmetric bell shape around mean μ
  • Mean = Median = Mode
  • Defined by μ and σ². Written X ~ N(μ, σ²)
  • Total area under curve = 1
  • 95% of data within 1.96 SDs of mean

Sketch: symmetric bell, label μ at peak, tails to ±∞

Student's t-Distribution
  • Symmetric bell — but with FATTER tails
  • Defined by degrees of freedom (df)
  • As df increases, approaches normal
  • Mean = 0. Used when population variance is unknown

Sketch: two bells on same axes — narrow (normal) and shorter/wider (t)

Chi-Square Distribution
  • Right-skewed — NOT symmetric
  • Only takes positive values (≥ 0)
  • Defined by degrees of freedom (v)
  • Mean = v

Sketch: starts at 0, peak, long right tail. Axis starts at 0.

F-Distribution
  • Right-skewed, only positive values (≥ 0)
  • Defined by TWO df: numerator k and denominator n−k−1
  • Used exclusively for the F-test
  • Always positive — ratio of two chi-square variables

Sketch: same shape as chi-square. Write both df values on sketch.

Small-Sample vs Large-Sample Properties

★
Appears as "state and describe two/three small/large sample properties." Know both sets.

Small-Sample (hold for any n)

PropertyMeaning
UnbiasednessE(β̂) = β. No systematic over/under estimation.
EfficiencySmallest variance among all linear unbiased estimators.
Linearityβ̂ is a weighted linear sum of Y values.

Large-Sample (as n → ∞)

PropertyMeaning
Consistencyβ̂ → β as n increases. More data = more accurate.
Asymptotic Normalityβ̂ is approximately normally distributed for large n.
Asymptotic EfficiencySmallest variance among all consistent estimators.

Joint Probability Tables

When you see a table of X values, Y values and probabilities, the examiner is asking one of five things.

Reference table used throughout this section:

Y \ X-2023
10.270.080.160
300.040.100.36

1. Joint Probability

🎯
Question wording: "Find P(X=2, Y=3)" / "Find the joint probability when X=2 and Y=3"

Method: Just locate the intersection. No calculation needed.

Example

Find P(X=2, Y=3)

Go to row Y=3, column X=2 = 0.10

2. Marginal Probability

🎯
Question wording: "Find f(x)" / "Find P(X=2)" / "Find the marginal distribution of X or Y"

For X: add DOWN the column

Find P(X=2)
= 0.16 + 0.10 = 0.26

For Y: add ACROSS the row

Find P(Y=1)
= 0.27 + 0.08 + 0.16 + 0 = 0.51
⚠
Always check: all marginal probabilities must sum to 1.0

3. Conditional Probability

🎯
Question wording: "Find P(X=2 | Y=3)" / "Find the conditional probability of X given Y"
P(X | Y) = P(X, Y) / P(Y) Joint probability ÷ marginal probability of the given variable
Find P(X=2 | Y=3)
Step 1 — Find P(X=2, Y=3): read cell = 0.10 Step 2 — Find P(Y=3): sum row Y=3 = 0 + 0.04 + 0.10 + 0.36 = 0.50 Step 3 — Divide: 0.10 / 0.50 = 0.20

Given Y=3, there is a 20% probability that X=2.

4. Expected Value from a Table

🎯
Question wording: "Calculate E(X)" / "Calculate the expected value of X"
E(X) = Σ [ X × P(X) ]
Using the marginal distribution of X from the table above

Step 1 — Get marginal probabilities (add down each column)

P(X=-2) = 0.27+0 = 0.27 P(X=0) = 0.08+0.04 = 0.12 P(X=2) = 0.16+0.10 = 0.26 P(X=3) = 0+0.36 = 0.36

Step 2 — Multiply each X by its P(X), then sum

(-2)(0.27) + (0)(0.12) + (2)(0.26) + (3)(0.36) = -0.54 + 0 + 0.52 + 1.08 E(X) = 1.06

5. Variance from a Table

🎯
Question wording: "Calculate Var(X)" / "Calculate the variance of X"
Var(X) = E(X²) − [E(X)]²
Using E(X) = 1.06 from above

Step 1 — Calculate E(X²): square each X first, then multiply by P(X)

(-2)²(0.27) + (0)²(0.12) + (2)²(0.26) + (3)²(0.36) = 4(0.27) + 0 + 4(0.26) + 9(0.36) = 1.08 + 0 + 1.04 + 3.24 E(X²) = 5.36

Step 2 — Apply the formula

Var(X) = 5.36 − (1.06)² = 5.36 − 1.124 Var(X) = 4.236

Continuous PDF Questions

When the exam gives you f(x) = [some function] where [lower] ≤ x ≤ [upper] → X is continuous → use integration.

⚠
Integration rule: ∫xⁿ dx = xⁿ⁺¹/(n+1). Add 1 to the power, divide by the new power.

Expected Value — Continuous

E(X) = ∫ x · f(x) dx [over valid range]
f(x) = x/3, 0 ≤ x ≤ 3 (Test 1 style)
E(X) = ∫₀³ x·(x/3) dx = (1/3) ∫₀³ x² dx = (1/3) · [x³/3] from 0 to 3 = (1/3) · (27/3 − 0) = (1/3) · 9 E(X) = 3
f(x) = x/3, 0 ≤ x ≤ 1 (Test 1 2026)
E(X) = ∫₀¹ x·(x/3) dx = ∫₀¹ x²/3 dx = [x³/9] from 0 to 1 = 1/9 − 0 E(X) = 1/9 ≈ 0.111

Variance — Continuous

Var(X) = E(X²) − [E(X)]² E(X²) = ∫ x² · f(x) dx — same method but square x first
  1. Find E(X) using the integral above
  2. Find E(X²) = ∫ x²·f(x) dx over the valid range
  3. Apply: Var(X) = E(X²) − [E(X)]²
f(x) = x/3, 0 ≤ x ≤ 1 — E(X) = 1/9

Find E(X²)

E(X²) = ∫₀¹ x²·(x/3) dx = ∫₀¹ x³/3 dx = [x⁴/12] from 0 to 1 = 1/12

Apply formula

Var(X) = 1/12 − (1/9)² = 0.0833 − 0.0123 Var(X) = 0.0710
— PRACTICAL — OLS Regression

OLS Regression

The core of every Test 2. Calculate β̂₁ first, always.

Slope Coefficient β̂₁

🎯
Question wording: "Calculate β₁" / "Compute the OLS estimates" / "Calculate the slope coefficient"
β̂₁ = Σ[(Xᵢ−X̄)(Yᵢ−Ȳ)] / Σ[(Xᵢ−X̄)²] If centred variables given (lowercase): β̂₁ = Σ(xᵢyᵢ) / Σ(xᵢ²)
⚠
Lowercase x and y = deviations from mean. UPPERCASE = raw values. Check what the exam gives you.

Intercept β̂₀

🎯
Question wording: "Calculate β₀" / "Calculate the intercept coefficient"
β̂₀ = Ȳ − β̂₁ · X̄ Calculate β̂₁ FIRST. Then compute means: X̄ = Σ(Xᵢ)/n, Ȳ = Σ(Yᵢ)/n

Interpretation of β̂₁

🎯
Question wording: "Interpret the slope coefficient estimate"
"A one-unit increase in [X] is associated with a [β̂₁] unit [increase/decrease] in [Y], ceteris paribus."
⚠
Always add "ceteris paribus" or "holding all other variables constant." Marks are specifically allocated for this.
Full Walkthrough — Test 2 (May 2024): Wages & Education, n=13

Given: Σ(xᵢyᵢ)=131.81, Σ(xᵢ²)=182, Σ(Yᵢ)=112.71, Σ(Xᵢ)=156

Calculate β̂₁

β̂₁ = 131.81 / 182 β̂₁ = 0.7242

Calculate means

X̄ = 156/13 = 12 Ȳ = 112.71/13 = 8.670

Calculate β̂₀

β̂₀ = 8.670 − (0.7242)(12) = 8.670 − 8.690 β̂₀ = −0.020

Write the equation

Ŷ = −0.020 + 0.7242·X

Interpret slope

A one year increase in schooling is associated with a R0.72 increase in mean hourly wages, ceteris paribus.

— PRACTICAL — Variance & Standard Error
Section F — Practical

Variance & Standard Error of β̂₁

You need these before you can do the t-test or confidence interval. Always compute in this order.

Step 1 — Error Variance σ̂²

σ̂² = Σ(eᵢ²) / (n − k − 1) Simple regression: k=1, so denominator = n−2. Multiple with 2 X's: n−3.
SymbolWhat it meansWhere you get it
Σ(eᵢ²)Sum of squared residualsGiven as Σ(eᵢ²), Σ(ûᵢ²), or RSS in the exam
nSample sizeStated in the question
kNumber of X variablesSimple regression = 1, multiple = count them

Step 2 — Variance of β̂₁

Var(β̂₁) = σ̂² / Σ(xᵢ²)

Step 3 — Standard Error of β̂₁

SE(β̂₁) = √Var(β̂₁) Always positive. This feeds directly into the t-statistic and confidence interval.
Test 2 (May 2024) continued — n=13, Σ(eᵢ²)=9.7173, Σ(xᵢ²)=182

σ̂²

σ̂² = 9.7173 / (13−2) = 9.7173 / 11 σ̂² = 0.8834

Var(β̂₁)

Var(β̂₁) = 0.8834 / 182 Var(β̂₁) = 0.004854

SE(β̂₁)

SE(β̂₁) = √0.004854 SE(β̂₁) = 0.0697

R², Correlation & Elasticity

R² — Coefficient of Determination

🎯
Question wording: "Calculate the coefficient of determination" / "Estimate R²"
R² = ESS/TSS OR R² = 1 − RSS/TSS Use whichever values are provided. Both give the same answer.
TermFormulaWhat it is
TSSΣ(Yᵢ − Ȳ)² = Σ(yᵢ²)Total variation in Y. Given as Σ(yᵢ²) in exams.
RSSΣ(eᵢ²)Unexplained variation. Given as Σ(eᵢ²) in exams.
ESSTSS − RSSExplained variation. Calculate if not given.
Test 2 (May 2024) — RSS=9.7173, TSS=105.18
R² = 1 − (9.7173 / 105.18) = 1 − 0.0924 R² = 0.9076

90.76% of the variation in wages is explained by years of schooling.

Correlation Coefficient r

🎯
Question wording: "Calculate the correlation coefficient" / "Calculate r"
r = ±√R² Sign = sign of β̂₁. Positive slope → positive r. Negative slope → negative r.
⚠
Always check the sign of β̂₁ first. If slope is positive, r is positive. If negative, r is negative. The number is always between −1 and +1.
R²=0.9076, β̂₁=0.7242 (positive)
r = +√0.9076 r = +0.953

Strong positive linear relationship between education and wages.

Elasticity

🎯
Question wording: "Calculate and interpret the elasticity of Y with respect to X"
Elasticity = β̂₁ × (X̄ / Ȳ) Percentage change in Y for a 1% change in X. Uses sample means.
ValueInterpretation
|Elasticity| > 1Elastic — Y responds MORE than proportionally
|Elasticity| = 1Unit elastic
|Elasticity| < 1Inelastic — Y responds LESS than proportionally
PAT 1 (2026): β̂₁=0.589, X̄=125, Ȳ=110
Elasticity = 0.589 × (125/110) = 0.589 × 1.136 Elasticity = 0.669

A 1% increase in Investment leads to a 0.669% increase in GDP. Since 0.669 < 1, GDP is inelastic with respect to Investment.

— PRACTICAL — The t-Test
Section H — Practical

The t-Test

Same 4-step structure every single time. Do not deviate from it.

Types of Tests

TypeH₀H₁Reject H₀ when
Two-tailedβ₁ = 0β₁ ≠ 0|t_calc| > t_critical
Right one-tailedβ₁ ≤ 0β₁ > 0t_calc > t_critical
Left one-tailedβ₁ ≥ 0β₁ < 0t_calc < −t_critical

The t-Statistic

t = β̂₁ / SE(β̂₁) df = n − k − 1. Simple regression: df = n−2.

The 4-Step Method

  1. State hypotheses: H₀: β₁=0 H₁: β₁≠0 (or one-tailed version)
  2. Calculate t-statistic: t = β̂₁ / SE(β̂₁)
  3. Find t_critical from t-table using df and significance level
  4. Decision: If |t_calc| > t_critical → Reject H₀. If not → Fail to reject H₀.
Test 2 (May 2024) — β̂₁=0.7242, SE=0.0697, df=11, α=5% two-tailed
H₀: β₁ = 0 H₁: β₁ ≠ 0 t = 0.7242 / 0.0697 = 10.39 t_critical (5%, df=11, two-tailed) = 2.201 |10.39| > 2.201 REJECT H₀

There is a statistically significant positive relationship between education and wages at the 5% level.

Limitations of the t-Test

★
PAT 1 Q2(c) asked for 3 limitations worth 6 marks. 2 marks per limitation.
LimitationExplanation
Tests one coefficient at a timeCannot test if multiple coefficients are jointly zero. Use the F-test for that.
Assumes normally distributed errorsIn small samples with non-normal errors, results can be unreliable.
Sensitive to multicollinearityWhen X variables are correlated, standard errors inflate and t-stats shrink — variables may look insignificant when they are not.
Statistical ≠ economic significanceWith large n, even tiny meaningless effects can be statistically significant.
Sensitive to outliersExtreme data points can distort estimates and make the test unreliable.
— PRACTICAL — Confidence Intervals
Section I — Practical

Confidence Intervals

🎯
Question wording: "Construct a 95% confidence interval" / "Construct a 90% confidence interval for β₁"
CI = β̂₁ ± t_critical × SE(β̂₁) 95% CI → t at 5% significance. 99% CI → t at 1%. 90% CI → t at 10%.
⚠
CI confidence level + significance level = 100%. 95% CI uses 5% t-table. 99% CI uses 1% t-table. 90% CI uses 10% t-table.
Test 2 (May 2024) — β̂₁=0.7242, SE=0.0697, df=11, 99% CI
t_critical (1%, df=11, two-tailed) = 3.106 Margin = 3.106 × 0.0697 = 0.2165 Lower = 0.7242 − 0.2165 = 0.5077 Upper = 0.7242 + 0.2165 = 0.9407 99% CI: (0.508, 0.941)

We are 99% confident the true slope lies between 0.508 and 0.941.

The Zero Shortcut

After calculating the interval, ask: does zero lie inside the interval?

✓ Zero NOT in interval
Example: (0.508, 0.941)
Zero is outside → Reject H₀. β₁ is significant.
✗ Zero IN interval
Example: (−0.50, 1.20)
Zero is inside → Fail to reject H₀. β₁ not significant.

Interpretation Template

"We are [95%] confident that the true slope coefficient lies between [lower] and [upper]."
⚠
Never say: "There is a 95% chance that β₁ lies in the interval." That is incorrect. The interval either contains the true value or it doesn't — we are 95% confident in our method, not the specific interval.

The F-Test

Tests ALL slope coefficients simultaneously. Appears in Test 2 every year.

🎯
Question wording: "Test the overall significance of the regression model" / "Conduct an F-test"
t-testF-test
TestsOne coefficient e.g. β₁Entire model — β₁, β₂, β₃ together
H₀β₁ = 0β₁ = β₂ = β₃ = 0 (ALL slopes = 0)
H₁β₁ ≠ 0At least one βᵢ ≠ 0
F = (ESS / k) / [RSS / (n − k − 1)] Numerator df = k. Denominator df = n − k − 1. Both needed for F-table.
  1. State H₀: β₁=β₂=...=βₖ=0 and H₁: at least one βᵢ ≠ 0
  2. State decision rule: Reject H₀ if F_calc > F_critical
  3. Find F_critical from F-table using α, df₁=k, df₂=n−k−1
  4. Calculate F = (ESS/k) / [RSS/(n−k−1)]
  5. Compare and state conclusion in words
Test 2 (May 2026): n=50, k=3, ESS=1500, RSS=900
H₀: β₁=β₂=β₃=0 ESS/k = 1500/3 = 500 RSS/(n−k−1) = 900/(50−3−1) = 900/46 = 19.57 F = 500/19.57 = 25.55 F_critical(5%, df₁=3, df₂=46) ≈ 2.81 25.55 > 2.81 REJECT H₀ at 5%

The model is overall statistically significant. At least one variable matters.

Lecture Board: n=130, k=4, ESS=40,000, RSS=4,000
F = (40000/4) / (4000/125) = 10000 / 32 = 312.5 F_crit(5%, df₁=4, df₂=125) = 2.45 F_crit(1%, df₁=4, df₂=125) = 3.48 312.5 > 2.45 → REJECT at 5% 312.5 > 3.48 → REJECT at 1%

Multiple Regression & Log Models

Log Model Interpretation

★
From your lecture board: Log:Log = %. Log:Lin = multiply by 100. Lin:Log = divide by 100. Lin:Lin = units.
ModelFormInterpretation of β̂₁
Lin-LinY = β₀ + β₁X1 unit ↑ in X → β̂₁ unit change in Y
Log-Logln(Y) = β₀ + β₁·ln(X)1% ↑ in X → β̂₁% change in Y. β̂₁ IS the elasticity.
Log-Linln(Y) = β₀ + β₁X1 unit ↑ in X → (β̂₁ × 100)% change in Y
Lin-LogY = β₀ + β₁·ln(X)1% ↑ in X → (β̂₁/100) unit change in Y

Dummy Variables

A dummy variable takes 0 or 1 to represent a qualitative category. It shifts the INTERCEPT for different groups.

Y = β₀ + β₁X + β₂D where D = {0 or 1} D=0 (base group): Ŷ = β₀ + β₁X. D=1: Ŷ = (β₀+β₂) + β₁X

β̂₂ = average difference in Y between the two groups, holding X constant.

⚠
Dummy Variable Trap: If 3 groups, include only 2 dummies. Never include all — creates perfect multicollinearity. Always leave one as the base/reference group.

CLRM Assumptions & OLS Properties

The 7 CLRM Assumptions

★
Q1 of EVERY Test 2: "State any FOUR assumptions of the CLRM." 4 marks, one each. Know at least 6.
#AssumptionWhat it says
1Linearity in parametersThe model is linear in β₀ and β₁.
2Zero mean error termE(uᵢ) = 0. The model is not systematically wrong.
3HomoscedasticityVar(uᵢ) = σ² for all i. Constant variance — errors don't fan out.
4No autocorrelationCov(uᵢ, uⱼ) = 0. Errors are not correlated with each other.
5X is non-stochasticX is fixed and uncorrelated with the error. Cov(Xᵢ, uᵢ) = 0.
6No perfect multicollinearityNo X is a perfect linear function of another.
7Normality of errorsuᵢ ~ N(0, σ²). Needed for valid t and F tests.

OLS Properties — BLUE

★
PAT 1 Q1(f): "Discuss 4 OLS properties" = 8 marks. State the property + explain it = 2 marks each.
PropertyFormulaPlain explanation
UnbiasednessE(β̂) = βOn average, estimates equal the true value. No systematic error.
EfficiencyVar(β̂) is smallestAmong all linear unbiased estimators, OLS is most precise.
Linearityβ̂ = weighted sum of YᵢThe estimator is a linear function of the Y observations.
Consistencyβ̂ → β as n→∞More data = estimates get closer to the truth.
BLUEGauss-Markov TheoremUnder CLRM assumptions, OLS is the Best Linear Unbiased Estimator.
